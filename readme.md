# detect-newline-cli

> Detect the dominant newline character

## Install

```
$ npm install --global detect-newline-cli
```

## Usage

```
$ detect-newline --help

  Usage
    $ detect-newline <string>
    $ cat unicorn.txt | detect-newline

  Example
    $ detect-newline "$(printf 'Unicorns\nRainbows')"
    \n
```
